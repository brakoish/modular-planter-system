# Modular Planter System

A parametric, hexagonal modular planter system designed for 3D printing.

## Features

- **Hexagonal modules** - Connect infinitely in any direction
- **Parametric design** - Adjust size, wall thickness, drainage
- **Stackable** - Vertical expansion with stacking clips
- **Drainage system** - Integrated water management
- **No supports needed** - Designed for FDM printing

## Files

- `planter-module.scad` - Main planter module
- `connector-clip.scad` - Hex connection system
- `stacking-clip.scad` - Vertical stacking clips
- `drainage-tray.scad` - Optional water collection tray
- `customizer.py` - Python script for batch generation

## Print Settings

- **Material:** PETG or PLA
- **Layer height:** 0.2mm
- **Infill:** 15-20%
- **Walls:** 3
- **Top/Bottom:** 4 layers
